# 🚪Escape Room

A simple, beginner-friendly browser-based escape room game with 3 fun puzzles to solve!

---

## 📁 Project Structure

```
easy-escaper/
├── index.html   → Game structure & layout
├── style.css    → All styling & animations
├── script.js    → Game logic & puzzle answers
└── README.md    → You're reading this!
```

---

## 🚀 How to Run

1. Extract the ZIP file
2. Open `index.html` in any modern browser (Chrome, Firefox, Edge)
3. No server needed — works completely offline!

---

## 🎮 How to Play

1. Click **LET'S GO!** on the start screen
2. Click on any object in the room to open its puzzle
3. Read the clue board at the bottom for hints
4. Solve all **3 puzzles** to unlock the exit door 🚪
5. Click the door to escape and see your time!

---

## 🧩 The 3 Puzzles

| Object | Type | Clue |
|--------|------|------|
| 📝 Sticky Note | Number code | Count sides of shapes |
| 📦 Treasure Chest | Multiple choice | Classic riddle |
| 🕐 Wall Clock | Number code | Simple multiplication |

> 💡 Each puzzle has a **"Show Hint"** button if you get stuck!
